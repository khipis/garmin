# BitochiRun

Monster escape horror game for Garmin smartwatches.

## How to Play

1. Enter the dungeon and scan for monsters with your radar
2. When a monster is found — RUN!
3. Survive the chase and escape
4. Face increasingly dangerous creatures

## Features

- Radar/sonar scanner display with sweep animation
- Multiple monster types with unique pixel art sprites
- Dungeon scenery: stone walls, corridors, torches, fog wisps
- Accelerometer-based movement during scan phase
- Blood drip effects scaling with danger intensity
- Stamina and speed mechanics during chase
- Footprint trail system
- Star sky and ambient backgrounds per scene
- HUD with progress bar, timer, and score
- Vibration feedback on encounters

## Controls

- **Tilt** — move during scan
- **Tap** — advance / restart

## Scenes

- Menu → Intro → Scan → Found → Run → Escape/Caught → Final
- Each scene has unique dungeon scenery and atmosphere
