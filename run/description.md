# Garmin Connect IQ Store Description

## Run – Endless Runner Watch Game

Jump, duck, and sprint through an endlessly scrolling obstacle course. How far can you go?

### Features
- Side-scrolling endless runner with increasing speed
- Jump and slide mechanics
- Obstacle patterns that require quick reflexes
- High score tracking
- Simple one-button gameplay — easy to pick up

### Controls
- **SELECT** — jump
- **DOWN** — slide / duck
- **BACK** — exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
