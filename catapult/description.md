# Garmin Connect IQ Store Description

## Catapult – Aim & Launch Watch Game

Set the angle, adjust the power, and launch your projectile at the target. Simple physics, satisfying results.

### Features
- Adjustable angle and launch power
- Moving or static targets at varying distances
- Physics-based projectile arc
- Score based on accuracy
- Quick-play design — fast rounds

### Controls
- **UP / DOWN** — adjust angle
- **SELECT** — set power / launch
- **BACK** — exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
