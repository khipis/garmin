# BitochiSkywalker

Space combat game for Garmin smartwatches.

## How to Play

1. Pilot your starfighter through space
2. Aim with wrist tilt (accelerometer)
3. Tap to fire lasers at imperial fighters
4. Destroy all enemies in each wave to advance

## Features

- 8 unique enemy ship types with detailed pixel sprites
- Accelerometer-based crosshair aiming
- Laser beam effects with hit flashes
- Shield system — absorbs damage before hull
- Enemy attack patterns and return fire
- Parallax starfield with nebula backgrounds
- Cockpit HUD overlay with targeting reticle
- Wave progression with increasing difficulty
- Screen shake on hits and enemy fire
- Vibration feedback

## Controls

- **Tilt** — aim crosshair
- **Tap** — fire laser / advance between waves

## Progression

- Multiple waves of enemies per session
- Enemy count and toughness scales per wave
- Shield regeneration between waves
- Score based on accuracy and kills
