# BitochiParachute

3D perspective parachute game for Garmin smartwatches.

## How to Play

1. Jump from the plane
2. Freefall — steer toward the landing zone
3. Open your chute before it's too late
4. Navigate through rings for bonus points
5. Land on the target for the best score

## Features

- 3D behind-the-player perspective view
- Ground rendered with perspective grid, terrain patches, roads, trees
- Freefall and parachute phases with different controls
- Accelerometer steering (tilt to move)
- Ring bonus system — fly through rings for extra points
- Wind effects on descent
- Landing zone target with distance scoring
- Altitude-based visual scaling
- Cloud layer with parallax depth
- Vibration feedback and screen effects

## Controls

- **Tap** — jump from plane / open chute / advance
- **Tilt** — steer left/right/forward/back

## Scoring

- Landing accuracy (distance from target center)
- Ring bonuses collected during descent
- Grade system based on landing precision
