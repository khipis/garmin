# Garmin Connect IQ Store Description

## 8 Ball Pool – Watch Billiards Game

Classic pool game on your wrist. Aim, shoot, and pocket the balls on a full-screen billiards table.

### Features
- Full-screen pool table rendered on your watch
- Aim and power control with watch buttons
- 8-ball rules — pot your group, then the 8-ball to win
- Single player mode against basic AI
- Clean, minimal UI optimised for small screens

### Controls
- **UP / DOWN** — adjust aim angle
- **SELECT** — set power / shoot
- **BACK** — return to menu

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
