# Bitochi8Ball

Magic 8-Ball game for Garmin smartwatches.

## How to Play

1. Shake your wrist to ask a question
2. Watch the shaking animation
3. Get your mystical answer with sparkle effects

## Features

- Accelerometer-based shake detection
- Multiple answer categories (positive, neutral, negative)
- Animated 8-ball with sparkle effects
- Vibration feedback on shake

## Controls

- **Shake** — ask a question
- **Tap** — return to ready state

## Tech

- Monkey C / Connect IQ
- Accelerometer input via `Toybox.Sensor`
