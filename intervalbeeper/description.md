# Garmin Connect IQ Store Description

## Interval Beeper – HIIT Training Timer

Work-rest interval timer designed to run without looking at the screen. The watch tells you everything through vibration.

### Features
- Configurable work interval, rest interval, and cycle count
- Distinct haptic patterns for every phase transition — no need to look
- WORK / REST states with large countdown display
- Precise timing with no drift
- Default quick-start: 30s work / 30s rest / 10 cycles

### Vibration Patterns
- **Start** → long strong vibration
- **WORK starts** → 1 short pulse
- **REST starts** → 2 short pulses
- **Last 5s of WORK** → pulse every second
- **Workout complete** → long + short + long pattern

### Controls
- **SELECT** — start / pause
- **Long press** — reset
- **UP / DOWN** — adjust settings

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
