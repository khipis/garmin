# Garmin Connect IQ Store Description

## Bomb Defuse – Wire Cutting Watch Game

The clock is ticking. Choose the right wire and defuse the bomb before time runs out.

### Features
- Fast puzzle gameplay with a countdown timer
- Multiple wires — only one is safe
- Increasing difficulty: more wires, less time
- Tense haptic feedback as time runs low
- Minimalist design — maximum tension

### Controls
- **UP / DOWN** — select wire
- **SELECT** — cut selected wire
- **BACK** — exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
