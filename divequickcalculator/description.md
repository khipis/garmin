## Emergency Dive Quick Calculator – MOD NDL

The fastest dive safety check on your wrist. Open the app — the result is already there. No menu, no loading, no confirmation screen. Just two inputs (gas + depth) and an instant SAFE / WARNING / DANGER verdict.

Designed for the surface, between dives, or pre-dive briefing. Not a dive computer.

---

### How It Works

The app opens **directly** on a live calculator. The background itself changes colour:

| Background | Meaning |
|---|---|
| 🟢 Dark green | PO2 ≤ 1.4 → **SAFE** |
| 🟠 Dark amber | PO2 1.4–1.6 → **WARNING** |
| 🔴 Dark red (pulsing) | PO2 > 1.6 → **DANGER** |

You read the verdict before you even read the text.

---

### Two Pages

**Page 1 — Gas Check** (opens first)

- Select gas: Air 21% / Nitrox 32 / Nitrox 36
- Select depth: 5–40m (13 steps)
- Instant output: PO2, SAFE/WARNING/DANGER verdict, MOD, NDL estimate

**Page 2 — Best Mix**

- Select depth
- Instant output: optimal FO2%, gas name (Nitrox XX), MOD, NDL

---

### Controls

| Button | Action |
|---|---|
| UP / DOWN | Change value of active field |
| SELECT | Toggle gas ↔ depth (Page 1) |
| MENU / ► ◄ | Switch between pages |
| BACK | Return to Page 1 or exit |
| TAP upper | Decrement value |
| TAP lower | Increment value |

---

### Calculations

- **PO2** = FO2 × (depth/10 + 1)
- **MOD** = (1.4 / FO2 − 1) × 10
- **NDL** = PADI RDP table with linear interpolation; nitrox uses EAD for automatic NDL extension
- **Best Mix** = 1.4 / (depth/10 + 1)

---

### Safety

This app is for informational and planning purposes only. It is **not** a dive computer. Always use a certified dive computer underwater. Follow your training. Always analyse your gas.
