# Garmin Connect IQ Store Description

## Ski Jump – Distance Watch Sports Game

Time your jump and land as far as possible. Classic ski jump gameplay with satisfying physics.

### Features
- In-run speed builds automatically — timing is everything
- Jump angle selection at the ramp edge
- Body position control during flight for maximum distance
- Landing telemark style for style points
- Distance-based scoring with personal best tracking

### Controls
- **SELECT** — jump at the ramp lip
- **UP / DOWN** — adjust body position mid-flight
- **BACK** — restart / exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
