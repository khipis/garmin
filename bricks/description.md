# Garmin Connect IQ Store Description

## Bricks – Breakout Arcade Watch Game

Classic Breakout/Arkanoid gameplay on your wrist. Bounce the ball, destroy bricks, don't let it fall.

### Features
- Multiple rows of colored bricks to destroy
- Paddle controlled with watch buttons or tap
- Ball speed increases as you clear bricks
- Level progression — new layouts with each level
- Score tracking

### Controls
- **UP / DOWN** — move paddle left/right
- **BACK** — pause / exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
