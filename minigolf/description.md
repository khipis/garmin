# Garmin Connect IQ Store Description

## Mini Golf – Putt Watch Puzzle Game

Miniature golf on your wrist. Aim, choose your power, and putt the ball into the hole.

### Features
- Multiple hole layouts with obstacles, ramps, and walls
- Angle and power control
- Stroke count tracking — aim for under par
- Increasing course difficulty
- Clean, top-down view of each hole

### Controls
- **UP / DOWN** — adjust aim direction
- **SELECT** — set power / putt
- **BACK** — exit / hole menu

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
