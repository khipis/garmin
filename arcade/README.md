# BitochiAxeArcade

Knife-hit style arcade game for Garmin smartwatches.

## How to Play

1. A circular log rotates in the center of the screen
2. Tap to throw an axe — it flies up and sticks into the log
3. Avoid hitting axes already stuck in the log
4. Survive as many throws as possible

## Features

- Rotating log with wood grain detail
- Angle-based collision detection (14° tolerance)
- Progressive difficulty — speed increases, direction reverses
- Level system — clear 5 axes per level to advance
- Wood chip and spark particle effects
- Screen shake and vibration feedback
- Night sky with twinkling stars and flickering torches

## Controls

- **Tap** — throw axe

## Progression

- Every 5 successful sticks clears the log and advances level
- Rotation speed increases each level
- Random direction changes add unpredictability
