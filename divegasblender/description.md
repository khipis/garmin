# Garmin Connect IQ Store Description

## Dive Gas Blender — Nitrox PP Blending Calculator

Plan and verify your nitrox fills using partial-pressure blending — directly on your wrist.

### Features

- **PP Blend** — calculate the exact O2 fill pressure and air top-up pressure for any target nitrox mix
- **Mix Check** — verify what FO2 you actually achieved after filling, based on real fill pressures
- **MOD Lookup** — instant Maximum Operating Depth table for any FO2 (at PO2 1.4 and 1.6 ATA)
- Supports blending into empty or partially-filled tanks with any existing mix
- Results shown instantly as you scroll through values — no buttons to press

### How PP Blending Works

Partial-pressure blending fills the tank in two steps:

1. Fill pure O2 to a calculated intermediate pressure
2. Top up with air (21% O2) to the target fill pressure

This tool calculates the exact Step 1 target pressure based on your:
- Target nitrox percentage (21–40% FO2)
- Target fill pressure (50–300 bar)
- Current pressure already in tank
- Existing mix already in tank

### Controls

- **UP / DOWN** — scroll through preset values
- **SELECT** — advance to next input field
- **BACK** — return to previous field / menu
- **TAP** — adjust value or navigate

### Safety Notice

Always verify blends with a calibrated O2 analyser before diving. This tool is for planning only — not a substitute for proper blending training and certification.
