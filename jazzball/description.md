# Garmin Connect IQ Store Description

## JazzBall – Ball Trap Watch Arcade Game

Draw walls to shrink the playfield and trap bouncing balls — without getting hit. Classic JezzBall-style gameplay on your wrist.

### Features
- Bouncing balls that you must isolate
- Draw horizontal or vertical dividers to cut off empty space
- Level complete when 75%+ of the area is cleared
- Multiple balls from level 2 onward
- Score based on cleared percentage

### Controls
- **UP / DOWN** — choose wall direction
- **SELECT** — start / stop drawing wall
- **BACK** — exit / menu

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
