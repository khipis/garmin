# BitochiSwing

Pendulum swing game for Garmin smartwatches.

## How to Play

1. Your character swings automatically on a vine
2. Tap to release at the right moment
3. Fly through the air using momentum
4. Catch the next anchor point to continue swinging

## Features

- Sine-wave pendulum physics with tangential velocity on release
- Projectile motion with gravity after release
- 3 anchor types: Tree Branch, Giant Mushroom, Crystal Formation
- Side-scrolling camera with parallax layers
- Fairy tale enchanted forest visuals
- Animated character sprite with swing/fly poses
- Magic sparkle and dust particle effects
- Layered sky with moon, stars, mountains, trees
- Ground with flowers, mushrooms, and fireflies
- Combo system for consecutive catches
- Distance scoring

## Controls

- **Tap** — release swing / start game

## Progression

- Anchor spacing increases with distance
- Swing speed and timing become more demanding
- Score based on total distance and combo catches
