# BitochiAxe

Viking-themed axe throwing game for Garmin smartwatches.

## How to Play

1. Aim by tilting your wrist (accelerometer)
2. Tap to throw the spinning axe
3. Axe must hit the target log with the blade at the correct angle
4. Avoid hitting previously stuck axes (stacking challenge)

## Features

- Realistic axe rotation physics with angular velocity
- Angle-based hit detection — blade must face target on impact
- Moving target in later rounds
- Accelerometer-based aim adjustment
- Viking-style pixel art: wooden planks, runes, torches, falling snow
- Combo system for consecutive perfect hits
- Wood chip and spark particle effects
- Screen shake and vibration feedback

## Controls

- **Tilt** — adjust horizontal aim
- **Tap** — throw axe

## Scoring

- Normal hit, Perfect hit (center), Blocked (hit existing axe)
- Combo multiplier for streaks
- Lives system with escalating difficulty per wave
