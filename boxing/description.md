# Garmin Connect IQ Store Description

## Boxing Timer – Round Counter Watch

Professional boxing / kickboxing / MMA round timer with haptic alerts. Train without looking at the screen.

### Features
- Configurable round duration, rest time, and number of rounds
- Full-screen color states: GREEN (active), RED (rest), YELLOW (last 10s)
- Strong haptic feedback at every transition — distinguishable patterns
- Large centered countdown + round indicator (e.g. "Round 3/5")
- Warning buzz in final 10 seconds of each round

### Controls
- **SELECT** — start / pause
- **Long press** — reset
- **UP / DOWN** — adjust settings

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
