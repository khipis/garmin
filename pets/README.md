# BitochiPets

Virtual pet game for Garmin smartwatches.

## How to Play

1. Choose your pet type from the selection screen
2. Feed, play with, and care for your pixel creature
3. Keep hunger, happiness, and health in balance
4. Play mini-games to earn rewards

## Features

- Multiple pet types with unique appearances
- Unlockable pets: Undead (unlock by losing a pet) and Dzikko (unlock by overfeeding)
- Pet lifecycle: aging, hunger, happiness, health
- Persistent pet state saved between sessions
- Mini-games:
  - **Memory** — card matching game
  - **React** — reaction time challenge
  - **Rush** — speed tapping game
  - **Dodge** — obstacle avoidance

## Controls

- **Tap** — interact / feed / play
- **Swipe/Buttons** — navigate menus

## Pet Care

- Feed regularly to prevent starvation
- Play games to keep happiness up
- Don't overfeed (unlocks Dzikko)
- Pet can die from neglect (unlocks Undead)

## Storage

Uses `Application.Storage` for persistent pet data and unlock flags.
