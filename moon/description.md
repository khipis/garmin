# Garmin Connect IQ Store Description

## Moon Lander – Classic Watch Game

Pilot your lunar lander to a safe touchdown. Control thrust and attitude — land gently or crash trying.

### Features
- Classic Lunar Lander physics with gravity and fuel
- Tilt/accelerometer control for lateral movement (where supported)
- Thrust button to fight gravity
- Landing pad that progressively shrinks — increasing difficulty
- Velocity indicator — stay under the landing speed limit
- Crash or safe landing animations

### Controls
- **SELECT / UP** — fire main thruster
- **DOWN / tilt** — lateral control
- **BACK** — restart / exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more. Accelerometer control on supported models.
