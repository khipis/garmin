# Garmin Connect IQ Store Description

## Blobs – Color Match Puzzle Watch Game

Match and merge colored blobs to clear the board. Simple mechanics, surprisingly deep strategy.

### Features
- Color-matching puzzle on a compact grid
- Chain reactions for high combos
- Increasing difficulty as the board fills
- Score tracking — beat your best
- Minimalist UI, no clutter

### Controls
- **UP / DOWN / SELECT** — navigate and select blobs
- **BACK** — pause / return to menu

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
