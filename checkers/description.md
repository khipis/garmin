# Garmin Connect IQ Store Description

## Checkers – Classic Board Watch Game

Full checkers game on your Garmin watch. Play against the built-in AI or use hot-seat two-player mode.

### Features
- Standard checkers rules including mandatory captures and king promotion
- Smart cursor navigation — select piece, pick destination
- AI opponent with basic strategy
- Two-player pass-and-play support
- Full 8×8 board rendered on watch screen

### Controls
- **UP / DOWN / LEFT / RIGHT (or swipe)** — move cursor
- **SELECT** — select / confirm move
- **BACK** — undo / exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
