# Garmin Connect IQ Store Description

## Sparring Timer – BJJ Boxing MMA Rounds

Professional combat sports round timer with full-screen color states and powerful haptic feedback. Train hard without looking at the screen.

### Features
- Configurable round duration, rest time, and number of rounds (defaults: 5 min / 1 min / 5 rounds)
- Full-screen bold colors — no guessing what phase you're in:
  - **GREEN** → active round
  - **RED** → rest period
  - **FLASHING YELLOW/GREEN** → last 10 seconds warning
- Large centered countdown and round indicator (e.g. "Round 3/5")
- Distinct vibration patterns for every transition

### Vibration Patterns
- **Round starts** → 1 strong vibration
- **Last 10s** → quick pulses every second
- **Round ends** → long strong vibration
- **Rest starts** → 2 short vibrations
- **Rest ends** → 1 strong vibration

### Controls
- **SELECT** — start / pause
- **Long press** — reset
- **UP / DOWN** — adjust settings

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
