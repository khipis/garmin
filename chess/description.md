# Garmin Connect IQ Store Description

## Chess – Classic Strategy Watch Game

Full chess on your wrist. Play against the built-in engine or challenge a friend with pass-and-play.

### Features
- Complete chess rules: castling, en passant, promotion, check/checkmate detection
- Smart cursor — highlights legal moves for selected piece
- AI opponent (basic heuristic engine)
- Two-player hot-seat mode
- Compact board rendering optimised for all watch screen sizes

### Controls
- **UP / DOWN / LEFT / RIGHT (or swipe)** — move cursor
- **SELECT** — select / place piece
- **BACK** — undo / exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
