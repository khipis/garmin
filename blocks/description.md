# Garmin Connect IQ Store Description

## Blocks – Tetris Style Watch Puzzle Game

The classic falling-block puzzle, reimagined for your Garmin watch. Clear rows, survive as long as you can.

### Features
- Classic Tetromino pieces falling on a full-screen grid
- Increasing speed as you progress
- Next piece preview
- Score and level tracking
- Responsive tap and button controls

### Controls
- **UP / DOWN** — move piece left/right
- **SELECT** — rotate piece
- **Long press** — hard drop
- **BACK** — pause / exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
