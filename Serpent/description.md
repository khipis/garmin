# Garmin Connect IQ Store Description

## Serpent – Snake Classic Watch Game

The timeless Snake game, redesigned for your Garmin watch. Eat, grow longer, don't hit the walls.

### Features
- Classic Snake gameplay on a full-screen grid
- Speed increases as the snake grows
- Wrap-around or wall-collision modes
- High score tracking
- Smooth directional control

### Controls
- **UP / DOWN / LEFT / RIGHT (or swipe)** — change snake direction
- **SELECT** — start / pause
- **BACK** — exit

### Compatibility
Fenix 5–8, Epix 2, Forerunner series, Instinct, Venu, Vivoactive and more.
